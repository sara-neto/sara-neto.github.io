// About me

Where data meets
clinical precision.

I spent five years as a Cardiac Physiologist, interpreting ECGs, Holter monitors, and stress tests — work that demands meticulous attention to signals in noisy data. That mindset transferred naturally into data analytics.

I'm now pursuing a Master's in Information Management (BI) at NOVA IMS, while concurrently completing Data Science and Machine Learning courses at Instituto Superior Técnico. My goal is to apply analytical rigor to healthcare and business data challenges.

My thesis evaluated the Samsung Galaxy Watch 4 as a clinical-grade ECG device — combining clinical expertise with signal processing, data collection, and statistical validation. A rare blend of domain knowledge and technical skill.

BI Project · NOVA IMS · Sep–Dec 2025

UFO Sightings — Enterprise Data Warehouse

Built a full enterprise BI architecture using UFO sightings data as a case study. Designed a layered Lakehouse → Staging → Data Warehouse stack using Kimball dimensional modelling principles. Defined fact table grain (one record per sighting), created surrogate keys and dimension tables, performed data cleaning and standardisation, and automated the full ETL workflow through Microsoft Fabric Pipelines and Dataflows.

Microsoft Fabric SQL Kimball Modelling ETL Pipelines Data Warehouse Dataflows

Data Mining Project · NOVA IMS · Sep–Dec 2025

Customer Segmentation via Unsupervised ML

Segmented a consumer dataset using unsupervised machine learning to identify distinct customer profiles for targeted marketing. Conducted comprehensive exploratory data analysis and data preparation, applied PCA for dimensionality reduction, and developed clustering models to extract actionable, data-driven marketing strategies tailored to each segment.

Python Clustering PCA EDA Unsupervised ML Marketing Analytics
